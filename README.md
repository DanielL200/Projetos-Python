# Projetos-Python
Projeto para uma Loja de Utensílios

Programação Orientada a Objetos:

1. Funções;
2. Estrutura de Decisões;
3. Operadores Matemáticos.

Objetivo do Sistema:

O objetivo é automatizar processos de uma loja de utensílios, como cálculo de lucro, aplicação de descontos e impostos, tornando as operações mais rápidas e eficientes.
